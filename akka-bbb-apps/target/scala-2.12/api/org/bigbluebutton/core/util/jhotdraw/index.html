  2.  class BezierPath extends ArrayList[Node] with Shape with Serializable with Cloneable

    BezierPath allows the construction of paths consisting of straight lines, quadratic curves and cubic curves.

    BezierPath allows the construction of paths consisting of straight lines, quadratic curves and cubic curves.

    A BezierPath is defined by its nodes. Each node has three control points: C0, C1, C2. A mask defines which control points are in use. At a node, the path passes through C0. C1 controls the curve going towards C0. C2 controls the curve going away from C0.

  3.  class BezierPathIterator extends PathIterator

    This class represents the iterator for a BezierPath.

    This class represents the iterator for a BezierPath. It can be used to retrieve all of the elements in a BezierPath. The BezierPath#getPathIterator method is used to create a BezierPathIterator for a particular BezierPath. The iterator can be used to iterator the path only once. Subsequent iterations require a new iterator.
